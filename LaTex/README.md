# Shared LaTeX Files

This folder is for reusable LaTeX infrastructure only.

- `preamble.tex`: shared packages and commands for the slide decks.
- `simplemetropolis.sty`: shared Beamer style.
- `img/`: images used by multiple LaTeX documents.
- `simpson/`: animation frames used by the linear regression slides.

Course documents themselves now live in their content folders, such as `1_SLIDES/`, `2_PROBLEM_SETS/`, `5_GROUP_PROJECT/`, and `9_EXAMS/`.
